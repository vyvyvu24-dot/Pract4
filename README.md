# Конвертер учебного конфигурационного языка (Вариант №7) в TOML

## Описание
Инструмент командной строки для преобразования текста на учебном конфигурационном языке (вариант №7) в формат TOML.

## Особенности языка (вариант №7)
- Однострочные комментарии: `# ...`
- Многострочные комментарии: `--[[ ... ]]`
- Числа: `[+-]?\d+\.?\d*[eE][+-]?\d+`
- Массивы: `'( значение значение значение ... )`
- Словари: `{ имя => значение, ... }`
- Имена: `[_a-zA-Z]+`
- Строки: `@"строка"`
- Константы: `имя: значение` и `|имя|`
